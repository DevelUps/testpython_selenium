los codigos que veran aqui son ejemplos de 
código Python para utilizar Selenium WebDriver. 
Selenium es un marco de pruebas automatizadas 
para aplicaciones web que permite simular 
la interacción de un usuario con un navegador web.

El código importa la biblioteca Selenium y luego crea una 
instancia del controlador de Google Chrome, 
proporcionando la ruta del archivo ejecutable del controlador de Chrome.


install selenium
>pip install selenium

> search web page driver avaible https://chromedriver.chromium.org/
> Download and select the version chromedriver_win32
